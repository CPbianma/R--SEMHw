本研究通过相关性分析，系统性地探讨了手机依赖与心理健康指标、手机使用功能与动机、以及人口学变量之间的关联。结果表明，手机依赖与个体心理健康状况存在显著关联。

首先，手机依赖总体水平与心理健康指标呈现出中等程度且具有高度统计学意义的正相关。具体而言，手机依赖总分与压力（r = 0.514, p < 0.001）、焦虑（r = 0.461, p < 0.001）及抑郁（r = 0.499, p < 0.001）均显著正相关。这表明，个体对手机的依赖程度越高，其体验到的心理困扰水平也相应越高。此外，手机依赖的三个核心维度——戒断症状、渴求性以及身心影响——彼此之间及与依赖总分之间均存在高度相关（r值介于0.626至0.939之间，p < 0.001），这验证了手机依赖构念的内在一致性。每日手机使用时间虽与依赖总分显著正相关（r = 0.427, p < 0.001），但其与压力、焦虑和抑郁的直接关联未达到显著性水平，暗示使用时长并非影响心理健康的直接原因。
其次，关于手机使用功能与动机的分析揭示了一个发现：在众多功能和动机变量中，仅有“体现个性”的使用动机与手机依赖总分呈现出显著的正相关（r = 0.341, p = 0.008）。这一关联在渴求性维度上尤为突出（r = 0.484, p < 0.001），表明那些倾向于通过手机使用来彰显自我身份或寻求社会认可的个体，更可能发展出强烈的手机渴求，并进而形成更高的依赖。与此形成对比的是，大多数手机功能（如打电话、发短信、拍照、娱乐等）与依赖的相关性均不显著。一个值得注意的反常发现是，“手机功能-上网”与手机依赖总分（r = -0.258, p = 0.049）及戒断症状（r = -0.287, p = 0.027）呈显著负相关，这或许反映了工具性、信息导向的手机使用模式（如上网搜索、工作学习）与以娱乐和社交认同为核心的依赖路径存在本质差异，前者可能更具目的性和可控性。
最后，在人口学变量方面，本研究未发现性别、是否独生子女、父母受教育程度、恋爱状况、手机使用年限或每月手机消费与手机依赖总体水平存在任何显著相关性。这一结果提示，手机依赖作为一种行为成瘾现象，其形成可能更多地受到心理动机和使用模式等个体内在因素的影响，而非基本的人口社会学背景所主导。
值得指出的是，本研究的样本均来源于同一高校班级，样本在年龄、教育背景和生活环境上具有较高的同质性，这虽然有利于控制某些外部变量，但也限制了研究结论的推广性与普遍性。本研究揭示的相关关系，尤其是“体现个性”动机的核心作用，其稳定性和外部效度有待在更广泛、更多样的群体中进行进一步验证。
综上所述，本研究的结论在于：手机依赖与心理健康问题密切相关，且其形成的关键风险因素并非使用时间的长短或特定功能，而在于个体是否怀有“通过手机使用来体现个性”这一内在心理动机。这一发现将干预的焦点从外在的行为限制引向了内在动机的疏导与健康身份认同的建立，为后续的预防与干预实践提供了重要的理论依据与探索方向。
