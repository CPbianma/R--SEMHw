信效度评价
1.信度评价
本研究采用Cronbach's α系数评价问卷内部一致性信度。参考标准为：
Cronbach's Alpha ≥ 0.9  : 信度极好
Cronbach's Alpha 0.8-0.9: 信度良好
Cronbach's Alpha 0.7-0.8: 信度可接受
Cronbach's Alpha 0.6-0.7: 信度勉强
Cronbach's Alpha < 0.6  : 信度不佳，需修订
本次测量结果显示，所有量表及子维度的信度水平均达到了可接受标准以上，具体结果如下表。
表1 量表各维度信度评价
结论：手机依赖量表 (α = 0.930) 和DASS-21量表 (α = 0.931) 的总体信度均达到了优秀水平。各子维度的信度系数均在0.793以上，表明两个量表在本研究中均具有良好的信度，测量结果是可靠且稳定的。

2.结构效度
本研究采用验证性因子分析（CFA）来检验问卷的结构效度，即测量数据与理论模型的拟合程度。
（1）手机依赖量表：本研究验证了“手机依赖”的“戒断症状-渴求性-身心影响”三维度二阶因子结构。模型拟合指数结果显示：CFI = 0.949，TLI = 0.936，SRMR = 0.071，RMSEA = 0.083。
评价：CFI和TLI均大于0.90，SRMR小于0.08，表明模型拟合良好；RMSEA处于可接受范围的临界值。综合来看，模型拟合可接受，数据结果支持手机依赖量表的三维度二阶因子结构。此外，绝大部分题项的标准化因子载荷均大于0.70，进一步证明了量表具有良好的结构效度。
（2）DASS-21量表：本研究验证了“DASS-21”的“压力-焦虑-抑郁”三维度二阶因子结构。结果显示，该模型的各项拟合指数：CFI = 0.679, TLI = 0.637, RMSEA = 0.159均未达到理想标准。
评价：模型拟合不理想。这很可能与本次研究的样本量较小（N=59）有关，导致复杂的模型参数估计不稳定。尽管如此，该量表的三个维度之间存在高度相关（如表2），符合其理论预期。

压力与焦虑的相关系数为 r = 0.666，压力与抑郁的相关系数为 r = 0.675，焦虑与抑郁的相关系数为 r = 0.744。这三个相关系数（r值）都大于0.65，属于高度正相关的范围。且它们的p值均小于0.001，达到了统计上的极显著水平。

结论：手机依赖量表的“戒断症状-渴求性-身心影响”三维度二阶因子模型拟合指数可接受 (CFI = 0.949, TLI = 0.936, RMSEA = 0.083)，数据结果支持该量表的三维度理论结构。综合来看，问卷具有良好的结构效度。DASS-21量表的结构效度在本研究的小样本数据中未得到理想验证，但维度间的强相关性依然为量表的聚合效度提供了有力的证据。

3.效度指标（效标效度）
3.1每日手机使用时长与手机依赖量表总分
本研究以“每日手机使用时长”为外部效标，计算其与“手机依赖量表总分”的皮尔逊(Pearson)相关系数，以检验问卷的效标效度。
SPSS分析结果显示，手机依赖总分与每日手机使用时长之间存在中等强度的显著正相关（r =0.427, p < 0.001）。
这一结果表明，手机依赖得分越高的个体，其每日实际使用手机的时间也显著越长。这完全符合理论预期，有效证明了本问卷能够测量出实际行为表现，具有良好的效标效度。

SPSS分析结果显示，手机依赖总分与压力维度总分之间存在中等强度的显著正相关（r = .514, p < .001）。

以“DASS-21焦虑维度得分”为外部效标，计算其与“手机依赖量表总分”的皮尔逊(Pearson)相关系数，以检验问卷的效标效度。

3.4抑郁维度得分与手机依赖量表总分
