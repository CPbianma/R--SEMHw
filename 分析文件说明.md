# 结构方程模型（SEM）分析文件说明

## 项目概述

本项目包含手机依赖与心理健康关系的问卷数据及结构方程模型分析脚本。

**研究主题**: 手机依赖对大学生心理健康（压力、焦虑、抑郁）的影响

## 快速开始

### 使用R进行分析（推荐）

1. **安装必要的R包**:
```r
install.packages(c("lavaan", "semPlot", "readxl", "psych", "ggplot2", "corrplot"))
```

2. **运行分析脚本**:
```r
source("SEM_准备分析.R")
```

3. **查看结果**:
   - 控制台会显示描述性统计、信度分析、相关性分析、CFA和SEM结果
   - 查看生成的PDF文件了解模型路径图
   - 阅读`SEM分析结果摘要.txt`了解主要发现

### 使用Python进行分析

1. **安装必要的Python包**:
```bash
pip install pandas numpy scipy scikit-learn semopy matplotlib seaborn openpyxl factor_analyzer
```

2. **运行分析脚本**:
```bash
python SEM_准备分析.py
```

3. **查看结果**:
   - 控制台会显示各项分析结果
   - 查看生成的PNG图片了解数据分布和模型结果
   - 阅读`分析报告摘要.txt`了解主要发现

## SEM模型建议

### 推荐模型: 二阶因子结构模型

**测量模型**:
```
手机依赖（二阶潜变量）
├── 戒断症状（一阶潜变量）
├── 渴求性（一阶潜变量）
└── 身心影响（一阶潜变量）

负性情绪（二阶潜变量）
├── 压力（一阶潜变量）
├── 焦虑（一阶潜变量）
└── 抑郁（一阶潜变量）
```

**结构模型**:
```
手机依赖 → 负性情绪（压力、焦虑、抑郁）
```

**控制变量**:
- 性别
- 年龄
- 每日手机使用时间

### 简化模型: 使用维度总分

如果样本量较小（当前N=59），可以使用简化模型：

**测量模型**:
```
手机依赖 =~ 戒断症状总分 + 渴求性总分 + 身心影响总分
负性情绪 =~ 压力总分 + 焦虑总分 + 抑郁总分
```

**结构模型**:
```
负性情绪 ~ 手机依赖 + 控制变量
```

## 模型拟合指数标准

良好的模型拟合应满足以下标准：

| 指数 | 可接受 | 良好 |
|------|--------|------|
| CFI  | ≥ 0.90 | ≥ 0.95 |
| TLI  | ≥ 0.90 | ≥ 0.95 |
| RMSEA | ≤ 0.08 | ≤ 0.06 |
| SRMR | ≤ 0.08 | ≤ 0.05 |

## 注意事项

### 样本量限制
- 当前样本量（N=59）相对较小，对于复杂的二阶因子模型可能不够充分
- **建议**:
  1. 使用简化模型（维度总分作为观测变量）
  2. 使用稳健估计方法（如MLR）
  3. 谨慎解释结果，注意置信区间

### 性别分布不均
- 女性占79.7%，样本代表性有限
- **建议**: 在讨论中说明样本特征，限制结论的推广性

### 横断面数据
- 本研究为横断面设计，相关关系不等于因果关系
- **建议**: 在解释结果时使用"关联"而非"影响"，除非有理论支持
