## 📊 作业1：SEM二阶因子模型分析

### 项目概述

使用结构方程模型（SEM）的二阶验证性因子分析（Second-Order CFA）方法，探讨手机依赖与负面情绪（DASS-21）的潜变量结构。

**完整说明请查看**：`hw1/README.md` 和 `hw1/完整SEM分析使用指南.md`

### 核心特点

- **样本量**：N = 59
- **分析方法**：二阶验证性因子分析（Second-Order CFA）
- **软件工具**：R (lavaan包) + Python (可视化)
- **模型类型**：二阶潜变量模型

### 主要研究发现

#### ✅ 信度分析
- 手机依赖总量表：**α = 0.929**（优秀）
- DASS-21总量表：**α = 0.937**（优秀）
- 所有子维度：α > 0.78（良好至优秀）

#### ✅ 模型结构
```
模型1：手机依赖（二阶潜变量）
├── 戒断症状（一阶潜变量）→ 6个观测指标
├── 渴求性（一阶潜变量）→ 3个观测指标
└── 身心影响（一阶潜变量）→ 4个观测指标

模型2：负面情绪/DASS-21（二阶潜变量）
├── 压力（一阶潜变量）→ 7个观测指标
├── 焦虑（一阶潜变量）→ 7个观测指标
└── 抑郁（一阶潜变量）→ 7个观测指标

整合模型：手机依赖 ←→ 负面情绪（r = 0.550***）
```

#### ✅ 相关性分析
- 手机依赖 ↔ 负面情绪：**r = 0.550***（p < .001）
- 手机依赖三维度间：r = 0.626-0.796（高度相关）
- DASS-21三维度间：r = 0.666-0.744（高度相关）

#### ✅ 回归分析
- 手机依赖可解释负面情绪**30.3%**的方差
- 对压力预测最强：R² = 0.264
- 对抑郁预测：R² = 0.249
- 对焦虑预测：R² = 0.212

## 📊 作业2：大学生手机依赖与心理健康关系研究

### 研究概述

基于问卷调查和统计分析，探讨大学生手机依赖与心理健康（压力、焦虑、抑郁）的关系，并构建预测模型。

**核心成果**：`hw2-心理行为问题问卷设计/report/研究报告-大学生手机依赖与心理健康关系的实证研究.md` ⭐

### 研究设计

- **调查对象**：武汉大学公共卫生学院2022级预防医学专业学生
- **样本量**：59份有效问卷（预期74份）
- **抽样方法**：普查/整体抽样（Whole Group Sampling）

### 测量工具

1. **青少年手机使用依赖自评问卷（SQAPMPU）**
   - 13个题项，5级评分制
   - 3个维度：戒断症状、渴求性、身心影响
   - 信度系数：α = 0.930（优秀）

2. **DASS-21心理健康量表**
   - 21个题项
   - 3个维度：压力、焦虑、抑郁
   - 信度系数：α = 0.931（优秀）

### 主要研究发现

1. **相关性分析**
   - 手机依赖与压力（r=0.514, p<0.001）
   - 手机依赖与焦虑（r=0.461, p<0.001）
   - 手机依赖与抑郁（r=0.499, p<0.001）

2. **影响因素**
   - 抑郁情绪（OR=17.11, p=0.0464）
   - 每日使用时间（OR=14.03, p=0.0084）

3. **预测模型**
   - Logistic回归模型准确率：94.92%
   - AUC值：0.9341（优秀）

### 数据分析方法

- 描述性统计分析
- Pearson相关分析
- Logistic回归分析
- 验证性因子分析（CFA）
- ROC曲线分析

### 统计软件

- SPSS Statistics
- Python/R（用于部分高级分析）
