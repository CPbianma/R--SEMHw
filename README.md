## 📊 作业2：大学生手机依赖与心理健康关系研究

### 研究主题

青少年手机依赖与心理健康（压力、焦虑、抑郁）的关系实证研究

### 研究对象

- **调查对象**：武汉大学公共卫生学院预防医学专业学生
- **样本量**：59份有效问卷
- **抽样方法**：整体抽样（Census）

### 测量工具

1. **青少年手机使用依赖自评问卷（SQAPMPU）**
   - 13个题项，5级评分制
   - 3个维度：戒断症状、渴求性、身心影响
   - 信度系数：α = 0.930（优秀）

2. **DASS-21心理健康量表**
   - 21个题项
   - 3个维度：压力、焦虑、抑郁
   - 信度系数：α = 0.931（优秀）

### 主要研究发现

1. **相关性分析**
   - 手机依赖与压力（r=0.514, p<0.001）
   - 手机依赖与焦虑（r=0.461, p<0.001）
   - 手机依赖与抑郁（r=0.499, p<0.001）

2. **影响因素**
   - 抑郁情绪（OR=17.11, p=0.0464）
   - 每日使用时间（OR=14.03, p=0.0084）

3. **预测模型**
   - Logistic回归模型准确率：94.92%
   - AUC值：0.9341（优秀）

### 数据分析方法

- 描述性统计分析
- Pearson相关分析
- Logistic回归分析
- 验证性因子分析（CFA）
- ROC曲线分析

### 统计软件

- SPSS Statistics
- Python/R（用于部分高级分析）
